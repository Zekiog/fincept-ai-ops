# Hard Limits

## Execution
- No live trading in MVP
- No automatic order placement without human approval
- No prompt-based risk overrides
- No hidden connectors
- No unlogged broker actions

## Data
- One canonical provider per data domain
- One research output per request cycle
- One approval record per execution attempt

## Complexity
- Max 6 agents
- Max 5 active connectors
- Max 1 primary UI surface
- Max 1 risk policy source
