# Decision Matrix & Go/No-Go

## Go Conditions
- License path clarified
- Research gate implemented
- Paper broker validated
- Audit logging active
- Approval workflow working
- Real provider adapter selected

## No-Go Conditions
- Live trading in MVP
- Legal review missing
- No audit trail
- Schema errors present
- Uncontrolled connectors

## Realization Scope

| Phase | Scope | Decision |
|---|---|---|
| 1 | Research + briefing + paper broker | Go (MVP) |
| 2 | Backtest + screening + approval gate | Go (MVP+) |
| 3 | Sandboxed broker integration | Pilot |
| 4 | Live trading | No-Go |
| 5 | Multi-broker, full automation | V2 |
